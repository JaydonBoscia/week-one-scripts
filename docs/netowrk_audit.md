# Network Audit Script

## Overview
This script gathers network configuration details and tests basic connectivity
to assist with diagnosing network issues.

## Features
- Identifies hostname and IP address
- Displays all network interfaces and configurations
- Performs connectivity testing using ICMP ping

## Use Cases
- Network troubleshooting
- Verifying connectivity on workstations and servers
- Baseline network documentation

## Skills Demonstrated
- Network administration
- Data communications
- Python scripting
- Systems analysis
