# IT Support & Networking Scripts Portfolio

This repository contains Python scripts and documentation demonstrating IT support,
network administration, and systems engineering skills. It is structured by weeks
to simulate progressive training and hands-on experience.

## Repository Structure

- **scripts/** – Automation and diagnostic scripts (named by week)
- **docs/** – Documentation, use cases, and skills demonstrated
- **outputs/** – Generated reports from scripts (ignored by Git)
- **test_files/** – Sample/test files for scripts (ignored by Git)

## Week Overview

- **Week 1**: User audit, network audit, log analyzer (Help Desk, User Support, Application Support)
- **Week 2**: Network discovery, subnet scanning, Windows Event Log parser (Network Administration, Data Communications)
- **Future Weeks**: Automation, PowerShell scripts, server monitoring, network diagrams
